# Landing Page Builder Template

## Quick Start

Read: **`AGENT.md`**

That file contains the complete workflow with 5 phases.

---

## Overview

This template creates high-converting landing pages using:
- AI-generated images with TikTok-style social proof bubbles
- Buyer research-driven copy using Russell Brunson frameworks
- Automated build and deployment

## Workflow Summary

| Phase | Time | What You Do |
|-------|------|-------------|
| 1. Research | 10 min | Analyze competitor, complete buyer research |
| 2. Images | 20 min | Generate 35 images with TikTok bubbles |
| 3. Config | 5 min | Fill product.config |
| 4. Copy | 10 min | Rewrite sales copy |
| 5. Build | 5 min | Deploy to Netlify |

**Total: ~50 minutes**
