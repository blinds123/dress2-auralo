# Sections Directory

This folder contains modular HTML sections that get combined to create index.html.

## Build Process

Run `./build.sh` from the root folder to:
1. Concatenate all sections into index.html
2. Replace {{placeholders}} with values from product.config
3. Deploy to Netlify

## Editing Sections

See `phases/4-copy.md` for instructions on rewriting copy in these files.
